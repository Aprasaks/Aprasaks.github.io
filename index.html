CodeLoungeV2 — 코드 플레이그라운드

JavaScript, React, Vite, Monaco Editor

  • 개요: HTML/CSS/JS 코드를 탭 전환하며 작성하면 실시간 미리보기가 동작하며 실시간으로 다른 사용자와 소통할 수 있는 웹사이트.
  • 구현 방식:
    • 코드 입력은 monaco-editor를 사용 → VSCode처럼 자동완성과 문법 강조 기능을 위해 적용.
    • HTML/CSS/JS 코드를 합쳐 <iframe>에 삽입 → 입력 즉시 미리보기 기능을 제공.
    • localStorage 활용 → 백앤드에 대한 지식이 없는 상태에서 코드를 자동저장하고 불러오는 작업을 하기위해 사용.
  • 아쉬운 점: 웹소켓을 활용해 실시간 채팅뿐 아니라 코드 동기화도 구현하고 싶었지만, 당시 웹소켓에 대한 이해 부족으로 구현하지 못했습니다. 추후 학습을 통해 실시간 협업 기능을 보완할 계획입니다.
